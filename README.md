# Kiwi

Kiwi is a very simple and opinionated Key-Value database.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     kiwi:
       github: your-github-user/kiwi
   ```

2. Run `shards install`

## Usage

```crystal
require "kiwi"
```
